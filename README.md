# Firmware QA Portfolio – All-in-One Printer

This repository demonstrates my approach to firmware and embedded system testing using a simulated all-in-one printer environment.

The goal is to validate system behavior end-to-end — from user input to firmware processing to final output — ensuring reliability, data integrity, and correct device behavior.

## Modules Covered
- Print
- Scan
- Copy
- Network
- Firmware Updates
- Error Handling & Recovery

## Key Focus
- Device ↔ backend communication
- Firmware validation
- Failure scenarios (power, network, hardware)
- System reliability and performance

This portfolio reflects real-world testing approaches I applied in smart metering, POS systems, and device-based platforms.

## System Flow

User → Control Panel / PC → Firmware → Device (Print/Scan) → Output
                                  ↓
                               Network
                                  ↓
                               Backend
